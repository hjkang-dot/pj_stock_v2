# 📊 100점 만점 기회형 초성장주 투자 평가 모델 기준서

본 문서는 주식 분석 및 선별 시스템(Stock Finder V2)에 전면 적용된 **100점 만점 5대 투자 평가 모델 (기회형 초성장주 전략)**의 상세 배점 규칙과 선별 필터링 구조를 명시합니다.

---

## 1. 1차 필터링 및 후보 선정 기준 (`is_candidate`)

평가 대상을 1차 선별하고 최종 투자 추천 후보군(`is_candidate = 1`)으로 지정할 때 적용되는 필터 기준입니다.

* **최종 추천 후보(Candidate) 지정 조건**:
  * 평가 점수 합산 완료 후, 다음 **6가지 조건**을 모두 충족하는 기업에만 추천 뱃지(`🌟 추천`)가 부여됩니다.
    1. **종합 투자 점수**가 **60.0점** 이상
    2. **당기순이익**이 0보다 큼 (흑자 기업)
    3. **자본총계**가 0보다 큼 (자본잠식이 아닌 기업)
    4. **시가총액**이 0보다 큼
    5. **전년 대비 매출액 성장률**이 **5.0%** 초과
    6. **자기자본이익률(ROE)**이 **8.0%** 이상

---

## 2. 부문별 세부 평가 기준 (총 100점 만점)

종합 점수는 다음 5가지 부문의 점수를 합산하여 계산합니다.

### ① 금융 재무 안정성 (최대 15점 한도)
성장 기업의 재정 위기나 단기 부도 가능성을 예방하기 위해 부채 비율과 유동성을 평가합니다. (안정 배당주 전략에 비해 부채 허용 범위를 소폭 완화)

* **부채비율 (Debt Ratio) 점수 (최대 8점)**:
  * $부채비율 \le 100\%$ : **8점**
  * $부채비율 \le 150\%$ : **5점**
  * $부채비율 \le 250\%$ : **2점**
  * 부채비율 $250\%$ 초과, 혹은 데이터 부재 : **0점**
* **유동비율 (Current Ratio) 점수 (최대 7점)**:
  * $유동비율 \ge 150\%$ : **7점**
  * $유동비율 \ge 100\%$ : **4점**
  * $유동비율 \ge 70\%$ : **2점**
  * 유동비율 $70\%$ 미만, 혹은 데이터 부재 : **0점**

### ② 본업 지속 성장성 (최대 35점 한도)
성장주 투자 전략의 핵심 부문으로, 매출액 및 이익의 강력한 폭발적 상승 여부를 측정합니다. (전년 대비 성장률 기준)

* **매출액 성장률 점수 (최대 15점)**:
  * 매출액 $30\%$ 이상 증가 : **15점**
  * 매출액 $15\%$ 이상 증가 : **10점**
  * 매출액 $5\%$ 이상 증가 : **5점**
  * 매출액 $0\%$ 이상 증가 (유지) : **2점**
  * 매출액 역성장 ($0\%$ 미만) 혹은 데이터 부재 : **0점**
* **영업이익 성장률 점수 (최대 10점)**:
  * 영업이익 $40\%$ 이상 증가 : **10점**
  * 영업이익 $20\%$ 이상 증가 : **7점**
  * 영업이익 $5\%$ 이상 증가 : **4점**
  * 영업이익 $0\%$ 이상 증가 (유지) : **2점**
  * 영업이익 역성장 ($0\%$ 미만) 혹은 데이터 부재 : **0점**
* **주당순이익(EPS) 성장률 점수 (최대 10점)**:
  * EPS $40\%$ 이상 증가 : **10점**
  * EPS $20\%$ 이상 증가 : **7점**
  * EPS $5\%$ 이상 증가 : **4점**
  * EPS $0\%$ 이상 증가 (유지) : **2점**
  * EPS 역성장 ($0\%$ 미만) 혹은 데이터 부재 : **0점**

### ③ 경영 마진 및 이익 효율성 (최대 25점 한도)
자본의 운용 효율성과 사업의 마진 파워가 우수한지 평가합니다.

* **ROE (자기자본이익률) 점수 (최대 15점)**:
  * $ROE \ge 25\%$ : **15점**
  * $ROE \ge 15\%$ : **12점**
  * $ROE \ge 10\%$ : **8점**
  * $ROE \ge 5\%$ : **4점**
  * ROE 5% 미만, 혹은 데이터 부재 : **0점**
* **순이익률 (Operating Margin) 점수 (최대 10점)**:
  * $순이익률 \ge 20\%$ : **10점** (※ 평가 시스템 상 당기순이익 / 매출액 수치를 적용)
  * $순이익률 \ge 10\%$ : **7점**
  * $순이익률 \ge 5\%$ : **4점**
  * $순이익률 \ge 2\%$ : **2점**
  * 순이익률 2% 미만, 혹은 데이터 부재 : **0점**

### ④ 성장 재투자 성향 (최대 15점 한도)
벌어들인 돈을 주주 환원(배당)하기보다는 기업 내에 유보하여 성장을 촉진하기 위한 재투자(R&D, 설비 확장 등) 재원으로 활용하는 성향을 우대합니다.

* **배당 성향 (Payout Ratio) 점수 (낮을수록 우대, 최대 10점)**:
  * $배당성향 \le 30\%$ (무배당 포함) : **10점**
  * $배당성향 \le 50\%$ : **5점**
  * $배당성향 \le 80\%$ : **2점**
  * 배당성향 80% 초과, 혹은 데이터 부재 : **0점**
* **당기순이익 흑자여부 점수 (최대 5점)**:
  * 당기순이익 > 0 (흑자 기업) : **5점**
  * 당기순이익 적자, 혹은 데이터 부재 : **0점**

### ⑤ 가성비 성장 가치 (PEG 비율) (최대 10점 한도)
가치 대비 성장 매력도(GARP, Growth At a Reasonable Price)를 평가하기 위해 PEG Ratio(PER / EPS 성장률)를 평가합니다.

* **PEG Ratio (주가수익성장비율) 점수 (최대 10점)**:
  * $PEG \le 1.0$ : **10점** (성장 속도 대비 매우 저렴한 가격)
  * $PEG \le 1.5$ : **7점**
  * $PEG \le 2.5$ : **4점**
  * PEG 2.5 초과, 적자/역성장으로 계산 불가 시 : **1점**
