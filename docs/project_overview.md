# 프로젝트 개요

## 목표
- 자동으로 주가, 재무 데이터를 수집하고 평가해서 후보 추천 및 테스트


## 시스템 주요 기능
1. DART/KRX 데이터 수집
2. 수집 데이터 정제
3. 전략별 평가 및 후보 추출
4. 후보 종목 가상 테스트
5. 향후 AI 기반 정성 평가

## 로드맵

### 1차 MVP
- DART 재무, 배당 데이터 수집 (진행중)
- KRX 데이터 수집- 종목 목록 (완료)
- KRX 데이터 수집- 주가 (완료)
- 데이터 정제
- 저평가 배당주 전략 평가 및 후보 추출

### 1차 이후 기능
- 추가 전략 만들기
- UI 구성
- AI 기반 정성 평가
