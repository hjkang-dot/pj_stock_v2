# 📊 100점 만점 저평가 배당주 투자 평가 모델 기준서

본 문서는 주식 분석 및 선별 시스템(Stock Finder V2)에 전면 적용된 **100점 만점 5대 투자 평가 모델**의 상세 배점 규칙과 선별 필터링 구조를 명시합니다.

---

## 1. 1차 필터링 및 후보 선정 기준 (`is_candidate`)

평가 대상을 1차 선별하고 최종 투자 추천 후보군(`is_candidate = 1`)으로 지정할 때 적용되는 필터 기준입니다.

* **성장주 필터 해제**: 
  * 성장성이 하나의 부문 점수(15점)로 정식 편입됨에 따라 기존의 1차 성장 강제 필터(매출/영업이익이 매년 연속 성장하는 기업만 평가)를 완전히 해제하여, 전 상장 배당 종목을 대상으로 다각도 분석 및 비교가 가능합니다.
* **최종 추천 후보(Candidate) 지정 조건**:
  * 평가 점수 합산 완료 후, 다음 **5가지 조건**을 모두 충족하는 기업에만 추천 뱃지(`🌟 추천`)가 부여됩니다.
    1. **종합 투자 점수**가 **60.0점** 이상
    2. **당기순이익**이 0보다 큼 (흑자 기업)
    3. **자본총계**가 0보다 큼 (자본잠식이 아닌 기업)
    4. **시가총액**이 0보다 큼
    5. **주당 배당금(DPS)**이 0보다 큼 (실제 배당금을 배분한 기업)

---

## 2. 부문별 세부 평가 기준 (총 100점 만점)

종합 점수는 다음 5가지 부문의 점수를 합산하여 계산합니다.

### ① 금융 재무 안정성 (최대 15점 한도)
기업의 재정 위기나 단기 부도 가능성을 배제하기 위해 부채 비율과 유동성을 평가합니다.

* **부채비율 (Debt Ratio) 점수 (최대 8점)**:
  * $부채비율 \le 50\%$ : **8점**
  * $부채비율 \le 100\%$ : **6점**
  * $부채비율 \le 200\%$ : **4점**
  * $부채비율 \le 400\%$ : **2점**
  * 부채비율 $400\%$ 초과, 혹은 데이터 부재 : **0점**
* **유동비율 (Current Ratio) 점수 (최대 7점)**:
  * $유동비율 \ge 200\%$ : **7점**
  * $유동비율 \ge 150\%$ : **5점**
  * $유동비율 \ge 100\%$ : **3점**
  * 유동비율 $100\%$ 미만, 혹은 데이터 부재 : **0점**

### ② 본업 지속 성장성 (최대 15점 한도)
기업의 사업 규모와 이익 규모가 정체되지 않고 꾸준히 우상향하는지 평가합니다. (전년 대비 성장률 기준)

* **매출액 성장률 점수 (최대 5점)**:
  * 매출액 $10\%$ 이상 증가 : **5점**
  * 매출액 $5\%$ 이상 증가 : **4점**
  * 매출액 $0\%$ 이상 증가 (유지) : **2점**
  * 매출액 역성장 ($0\%$ 미만) 혹은 데이터 부재 : **0점**
* **영업이익 성장률 점수 (최대 5점)**:
  * 영업이익 $15\%$ 이상 증가 : **5점**
  * 영업이익 $5\%$ 이상 증가 : **4점**
  * 영업이익 $0\%$ 이상 증가 (유지) : **2점**
  * 영업이익 역성장 ($0\%$ 미만) 혹은 데이터 부재 : **0점**
* **주당순이익(EPS) 성장률 점수 (최대 5점)**:
  * EPS $15\%$ 이상 증가 : **5점**
  * EPS $5\%$ 이상 증가 : **4점**
  * EPS $0\%$ 이상 증가 (유지) : **2점**
  * EPS 역성장 ($0\%$ 미만) 혹은 데이터 부재 : **0점**

### ③ 가격 매력도 / 저평가 (최대 25점 한도)
기업의 이익 가치와 자산 가치에 비해 주가가 저렴한 매력적인 매수 시점인지 평가합니다.

* **PER (주가수익비율) 점수 (최대 15점)**:
  * $PER \le 6$배 : **15점**
  * $PER \le 10$배 : **12점**
  * $PER \le 15$배 : **8점**
  * $PER \le 25$배 : **4점**
  * PER 25배 초과, 0 이하, 혹은 데이터 부재 : **0점**
* **PBR (주가순자산비율) 점수 (최대 15점)**:
  * $PBR \le 0.6$배 : **15점**
  * $PBR \le 1.0$배 : **12점**
  * $PBR \le 1.5$배 : **8점**
  * $PBR \le 2.5$배 : **4점**
  * PBR 2.5배 초과, 0 이하, 혹은 데이터 부재 : **0점**
* **상한선 규정**: PER 점수와 PBR 점수를 합산한 결과에 `min(score, 25)`를 취해 최대 **25점**으로 제한합니다.

### ④ 주주 환원 및 배당 매력 (최대 25점 한도)
기업이 주주에게 성실히 이익을 나누어주는지, 그 분배 정책의 안정성과 지속성을 평가합니다.

* **배당 성향 (Payout Ratio) 점수 (최대 15점)**:
  * $20\% \le 배당 성향 \le 60\%$ : **15점** (가장 안정적이고 건강한 배당 성향)
  * $10\% \le 배당 성향 \le 80\%$ : **10점**
  * 그 외 구간 (10% 미만 혹은 80% 초과로 무리한 배당) : **5점**
  * 0 이하 혹은 데이터 부재 : **0점**
* **배당 유지/지속 연수 점수 (최대 15점)**:
  * 3년 이상 연속 현금배당 지급 : **15점**
  * 2년 연속 현금배당 지급 : **10점**
  * 1년 연속 (당해년도 최초 지급) : **5점**
  * 배당 미지급 : **0점**
* **자사주 가점 (Bonus, 최대 +3점)**:
  * 최근 자사주 매입/소각 진행 등 주주가치 제고 공시 이력 존재 시 가산 (※ 데이터 확보 예정으로 현재는 0점 가산 적용)
* **배당 삭감 감점 (Penalty, 최대 -15점 감점)**:
  * 최근 배당 삭감 이력 존재 시 (감액 횟수 $\times$ **-5점** 감점)
* **상한선 규정**: (배당 성향 + 배당 유지 + 자사주 가점 - 삭감 감점) 점수에 `max(min(score, 25), 0)`를 적용하여 **0점 ~ 25점** 범위로 제한합니다.

### ⑤ 시장 및 지배구조 리스크 (거버넌스) (최대 20점 한도)
* **주의**: 현재 거버넌스 부문은 지배구조, 경영진 주주 환원 의지 공시 데이터의 1차 확보 단계로서 **일괄 0점 고정 처리**되어 있습니다.
* **향후 계획**: 추후 LLM(거대언어모델)을 통한 공시 요약 데이터 및 정성적 분석 자료, 혹은 밸류업 프로그램 예시를 수집·가공하여 해당 지표에 20점 한도로 실데이터를 탑재할 예정입니다.
  * 상장 시장 배점 (KOSPI 5점, KOSDAQ 3점)
  * 시가총액 규모 배점 (최대 5점)
  * 거래 대금 배점 (최대 5점)
  * 상장 기간 배점 (최대 5점)
